# Spec 005: Page - Game Details Layout

## Overview
The structural layout for an individual game's page, combining catalog data and user interaction.

## Layout Rules (Grid)
- **Left Column (approx 25%):** Game cover (`GameCard` without hover), aggregated global rating, and user action box ("Log, rate or review").
- **Right Column (approx 75%):** Title, release year, developers, synopsis, and genres list.

## Data Requirements
Requires a mix of static catalog data (Spec 001) and dynamic user context (Review/Log).
